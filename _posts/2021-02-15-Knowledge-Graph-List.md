# 지식 그래프에 딥러닝 알고리즘 사용한 논문  

1번은 지식 그래프에 딥러닝을 적용한 Base?
2, 3, 4는 지식 그래프에 Convolution을 적용한 논문
5번은 지식 그래프에 R-GCN을 적용한 논문 - 새로운 방법을 알 수 있을 것 같아서

1. NTN: Reasoning With Neural Tensor Networks for Knowledge Base Completion  
링크 : https://www.aclweb.org/anthology/N18-2053.pdf  
2. ConvKB: A Novel Embedding Model for Knowledge Base Completion Based on Convolutional Neural Network  
링크 : https://www.aclweb.org/anthology/N18-2053.pdf    
3. Convolutional 2D Knowledge Graph Embeddings(ConvE)  
링크 : https://aaai.org/ocs/index.php/AAAI/AAAI18/paper/viewFile/17366/15884    
4. InteractE: Improving Convolution-based Knowledge Graph.  
링크 : https://arxiv.org/pdf/1911.00219.pdf    
5. Modeling Relational Data with Graph Convolutional Networks(R-GCN 적용)  
링크 : https://arxiv.org/pdf/1703.06103.pdf   
R-GCN을 적용하는 것도 찾아서 읽어보면 도움이 될 것 같음.
