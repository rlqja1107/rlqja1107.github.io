# Deep Informax
> 본 논문을 개인의 해석 및 이해를 바탕으로 정리 


## 논문원본  
[DIM](https://arxiv.org/abs/1808.06670)   

## 논문요약  

<h3 style="color: royalblue; font-weight: bold">논문 목적</h3>  

Deep Neural Network의 output과 input의 **Mutual Information**을 최대화하여 representation을 unsupervised 방식으로 학습하고자 하는 것이 목표.  

<h3 style="color: royalblue; font-weight: bold">Mutual Information</h3>  
Mutual Information의 형식적 의미는 Variable X, Y가 공유하는 정보량이다.   
$$P_{XY}$$와 $$P_{X}P_{Y}$$의 정보량 차이를 의미한다. X와 Y가 서로 무관하여 독립이면 0이 나온다. Mutual Information(MI)를 최대화하는 것은 X와 Y를 서로 dependent하게 만들면서 공유 정보량을 늘리는 것과 같다.  
식으로 보면 다음과 같다.  

$$I(X;Y) = \mathbb{E_{P_{XY}}} [log\frac{P_{XY}}{P_{X}P_{Y}}] = D_{KL}(P_{XY}\parallel P_{X}P_{Y})$$  

<h3 style="color: royalblue; font-weight: bold">DeepInformax의 목표</h3>  

1. feature map($$f_{\psi}$$)과 Representation $$Y$$의 MI를 최대화시킨다.  
2. Pushforward 분포($$Q_{\psi, X}$$)와 prior($$Q_{prior}$$)를 비슷하게 만든다.  

* Pushforward 분포를 prior과 유사하게 함으로써 $$Y$$가 특정한 통계 특성을 갖도록 하는 것이다. 논문에서는 independence를 예시로 들고 있다. independence의 특성을 갖게하여 Representation이 더 많은 의미있는 feature를 갖도록 하는 것이다.  

Independence하지 않는다면 2개 이상의 feature가 섞여서 생긴 Redundant feature가 많아지기 때문이다.  
<h3 style="color: royalblue; font-weight: bold">Global Informax</h3>  

Global Informax를 보기 전에 이미지를 예로 들었을 때 구조는 다음과 같다.  
<img width = "500" src = "https://user-images.githubusercontent.com/55014424/110619545-4af01100-81db-11eb-969c-c50885f555ac.png">  

X의 이미지가 있을 때, convolution을 통해 feature map $$f_{\psi}(x)$$를 형성한다. $$f_{\psi}(x)$$는 $$M \times M$$구조를 갖는다. 예시로 $$f_{c}$$의 Full-Connected Layer를 거쳐서 $$Y$$의 Representation을 갖는다.  

Global Informax의 경우에는 Feature map과 $$Y$$의 MI를 최대화시킨다. 해석했을 때, feature map의 정보가 최대한 Represenation에게 남아있도록, 공유되도록 하기 위함이라고 생각했다.  

**Donsker-Varadhan** Representation을 통한 Lowerbound   

$$I(X;Y) = D_{KL}(\mathbb{J} \parallel \mathbb{M} )>= \widehat{I_{w}}^{DV}(X;Y)$$  

$$\widehat{I_{w}}^{DV}(X;Y) = \mathbb{E}_{\mathbb{J}}[T_{w}(x,y)] - log\mathbb{E}_{\mathbb{M}}[e^{T_{w}(x,y)}]$$  

$$T_{w}(x,y)$$는 neural network로 구성된 discriminator function이다.

* Loss Function  

$$max_{w,\psi}\widehat{I}^{DV}(f_{\psi}(X);enc_{\psi}(X))$$   

$$ => max_{w,\psi}\mathbb{E}_{P_{XY}}[T_{w}(f_{\psi}, enc_{\psi}(x))]-log\mathbb{E}_{P_{X}X\tilde{P_{X}}}[e^{T_{w}(f_{\psi}(\tilde{x}),enc_{\psi}(x))}]$$  

#### Algorithm  

1. positive image $$x_{+}$$를 n개 뽑고, convolution을 이용하여 feature map $$f_{\psi}(x_{+})$$을 계산한다.  
2. $$y^{i} = h_{\psi}(f_{\psi}(x_{+}^{i})$$를 계산하여 Representation으로 표현한다.  
3. $$f_{\psi}(x_{+}^{i}), y^{i}$$의 positive pair를 맺는다. feature map과 feature map으로 나온 representation을 pair로  
4. Negative image $$x_{-}$$를 똑같이 n개 선택하고, feature map $$f_{\psi}(x^{i}_{-})$$을 계산한다.  
5. {$$f_{\psi}(x^{i}_{-}, y^{i})$$}의 쌍을 맺는다.  
6. $$\frac{1}{n} \sum^{n}_{i=1}T_{w}(f_{\psi}(x^{i}_{+}), y^{i}) - log\frac{1}{n}\sum^{n}_{i=1}e^{T_{w}(f_{\psi}(x^{i}_{-}),y^{i})}$$를 최대화시킨다.  

<h3 style="color: royalblue; font-weight: bold">Local Informax</h3>  

Global Informax만을 고려하게 된다면 문제점이 발생한다.  
> pixel size의 noise가 쓸모없게 된다. pixel에 noise가 발생했을 때, global informax는 전체적인 feature만을 고려하기 때문에 이를 고려할 수 없다.  

pixel size의 noise도 고려하기 위해서 image의 patch와 representation의 Mutual Information의 **평균**을 최대화시킨다.  

$$max \frac{1}{M^{2}} \sum^{M^{2}}_{j=1}\widehat{I}^{DV}(f_{\psi}(x)_{j}; enc_{\psi}(x))$$  

각 Patch의 정보를 살려 patch size의 noise도 감지하게 한다.  

#### Algorithm  

1. positive image $$x_{+}$$를 n개 뽑고, convolution을 이용하여 feature map $$f_{\psi}(x_{+})$$을 계산한다.  
2. $$y^{i} = h_{\psi}(f_{\psi}(x_{+}^{i})$$를 계산하여 Representation 표현한다.  
3. 한 Positive Item에 대해 {$$(f_{\psi}(x_{+}^{i})_{j}, y^{i})$$}로 pair시킨다. j는 한 image의 patch를 나타낸다. j = 1,2,....$$M \times M$$   
4. Negative image $$x_{-}$$를 똑같이 n개 선택하고, feature map $$f_{\psi}(x^{i}_{-})$$을 계산한다.  
5. {$$f_{\psi}(x^{i}_{-})_{j}, y^{i}$$}의 쌍을 맺는다.  
6. $$\frac{1}{M^{2}} \sum^{M^{2}}_{j=1} (\frac{1}{n} \sum^{n}_{i=1}T_{w}(f(x^{i}_{+}), y^{i}) - log\frac{1}{n} \sum^{n}_{i=1}e^{T_{w}(f(x^{i}_{-}),y^{i})})$$를 최대화시킨다.  

6번에서는 Patch와 Representation의 MI의 평균을 최대화시킨다. DV를 이용하여 MI의 lower bound를 최대화시켜 Estimate하면서 Maxmimize한다.  

<h3 style="color: royalblue; font-weight: bold">Matching Prior</h3>  

Pushforward 분포와 Prior분포를 가깝게하여 Representation이 통계적 특성을 갖도록 한다. Pushforward 분포의 경우에는 Representation의 분포로 생각해도 괜찮다. Adversarial Training으로 **GAN**의 방식과 비슷하다.  

$$\underset{\psi}{min}\underset{\theta}{max} D_{JSD}(Q_{prior} \parallel Q_{\psi,X})$$  

$$=> \underset{\psi}{min}\underset{\theta}{max} \mathbb{E}_{Q_{prior}}[logD_{\theta}(u)] + \mathbb{E}_{Q_{\psi,X}}[log(1-D_{\theta}(y))]$$  

GAN과 비슷하게, $$Q_{\psi,X}$$에서의 y는 Discriminator에서 최대값을 갖도록 한다. $$log(1-D_{\theta}(y))$$를 최소화해야하기 때문이다. Discriminator는 prior와 Pushforward에서 나온 sample를 최대한 구별하도록 학습시키고, Encoder는 output이 prior의 분포와 유사하도록 하기위해 $$D_{JSD}$$를 최소화시킨다. 

논문 강의에서는 prior가 gaussian보다 uniform일 때 더 잘 작동한다고 한다.
